# From GitHub Releases

1. Download the latest release from [Releases](https://github.com/alphingj/ru-share-media/releases)
2. Extract:
   ```bash
   tar -xzf ru-share-media.tar.gz
   ```
3. Run:
   ```bash
   ./ru-share-media
   ```

Default credentials: `admin` with a random password (shown in logs on first run).
